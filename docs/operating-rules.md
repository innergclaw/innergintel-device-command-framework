# Operating Rules

## 1. GitHub Is The Source Of Truth

If a project matters, it should have a GitHub repo.

Use GitHub for:

- client demo sites
- InnerG Intel skills
- docs
- reusable tools
- productized workflows

## 2. Telegram Is The Command Bus

Telegram is for updates, reports, and quick action prompts.

Use Telegram for:

- nightly business notes
- finance updates
- manual trade reminders
- client lead summaries
- "what changed?" reports

## 3. iPhone Is For Approval And Capture

Use the iPhone when speed or human confirmation matters.

Best for:

- quick decisions
- Telegram approval buttons
- mobile screenshots
- client message capture
- voice notes

## 4. MacBook 1 Builds

MacBook 1 is the primary production lane.

Best for:

- coding
- design builds
- local server checks
- GitHub deploys
- file editing

## 5. MacBook 2 Reviews

MacBook 2 is the second brain.

Best for:

- checking work
- comparing deployed versions
- running dashboards
- monitoring finance
- testing without interrupting builds

## 6. Sensitive Actions Stay Human-Confirmed

Codex can prepare and document sensitive actions, but final approval belongs to Nasir.

Sensitive actions include:

- live crypto trades
- live payments
- client commitments
- DNS changes
- repo deletion
- public launch announcements
